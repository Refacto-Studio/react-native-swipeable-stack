# Example App

This is an example Expo app demonstrating the usage of `react-native-swipeable-stack`.

## Running the Example

1. Navigate to the example folder:
```bash
cd packages/react-native-swipeable-stack/example
```

2. Install dependencies:
```bash
npm install
```

3. Run the app:
```bash
# Start Expo dev server
npx expo start

# Or run directly on iOS simulator
npx expo start --ios

# Or run directly on Android emulator
npx expo start --android
```

## What's Demonstrated

- ✅ Basic swipe left/right functionality
- ✅ Custom LIKE/NOPE overlays with rotation
- ✅ Imperative controls (buttons to trigger swipes)
- ✅ Undo functionality
- ✅ Custom card styling with images from Unsplash
- ✅ Empty state handling with reset button
- ✅ Spring physics for natural feel
